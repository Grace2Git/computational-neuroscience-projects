Computational Neuroscience Projects

This repository contains a collection of small computational neuroscience projects implemented in Python. 
The goal of these projects is to explore the dynamics of biologically inspired neuron models and neural networks.

Current Projects

1. Leaky Integrate-and-Fire (LIF) Neuron
Implementation of a classic spiking neuron model including:
- Euler numerical simulation
- Exact solution comparison
- Membrane potential dynamics
- Spike generation and refractory period

2. Stochastic LIF Neuron
Extension of the LIF model that incorporates Gaussian noise to simulate stochastic neural firing.  
This helps explore how noise affects spike timing and neural variability.

3. Neural Dynamics and Phase Plane Analysis
Visualization of neuron dynamics using phase-plane plots and parameter sweeps.

4. Simple Recurrent Neural Networks
Small RNN simulations exploring temporal dynamics in recurrent neural circuits.

Tools
- Python
- NumPy
- Matplotlib

Status
These projects are actively being expanded as part of an independent study in computational neuroscience.

Extensions:
- F-I curve analysis
- Parameter sweeps (tau, resistance, threshold)
- Phase-plane visualization of neuron dynamics
